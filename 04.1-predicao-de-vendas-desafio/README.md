# Regressão linear para prever vendas
Tentativa de fazer um algoritmo de predição de vendas utilizando Scikit-Learn com um algoritmo de regressão linear.
## Tecnologias
1. Python 3.11.0
2. Scikit-Learn
3. Pandas
4. Matplotlib
5. Seaborn
### Como rodar em ambiente virtual
```bash
pipenv sync
pipenv shell
python main.py
```
### Como parar e apagar o ambiente virtual
```bash
cntrl + d
pipenv remove
```
