# K-Means

## Resumo
Uma empresa apresenta dados de seus clientes, que também são empresas, como idade da empresa, faturamento mensal, nível de inovação, etc. Para dar um atendimento adequado a cada tipo de cliente, eles querem agrupar esses clientes conforme suas características.

Há uma construção de u algoritmo de clusterização que agrupe os clientes em segmentos.
