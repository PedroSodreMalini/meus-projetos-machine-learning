# Análise Exploratória de Dados da Netflix
Projeto com dataset da netflix para:
1. Mostrar tipos de dados disponíveis.
2. Ver o período da análise feita.
3. Determinar o tamanho da base de dados.
4. Verificar dados nulos.
5. Encontrar outliers.
## Tecnologias
1. Python 3.11.0
2. Pandas
3. Scipy
4. Matplotlib
